# Student Management System

## Description
A console-based Java application to manage student records efficiently. This project allows users to perform operations like adding, viewing, searching, and deleting student data.

It demonstrates the use of core Java concepts such as Object-Oriented Programming (OOP), ArrayList, and basic data structures. The project is designed to simulate real-world data handling and improve problem-solving skills.

---

## Features
- Add new student records
- View all student details
- Search student by ID
- Delete student records
- Simple and user-friendly console interface

---

## Tech Stack
- Java
- OOP Concepts
- ArrayList

---

## How to Run

1. Clone the repository:
   git clone https://github.com/yourusername/student-management-system.git

2. Navigate to the project folder:
   cd student-management-system

3. Compile the program:
   javac Main.java

4. Run the program:
   java Main
